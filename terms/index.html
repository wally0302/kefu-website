三、退換貨政策

1. 法律性質說明

依據《消費者保護法》及《通訊交易解除權合理例外情事適用準則》，本平台所提供之服務屬於「非以有形媒介提供之數位內容及線上即時運算服務」，其性質為一經開始提供即屬履行中之數位服務。

2. 七日解除權之說明與限制

用戶於購買前已明確知悉並同意：

  • 本服務為數位線上即時運算服務。
  • 當用戶實際使用點數，或系統依用戶指示開始提供 AI 運算、分析或回覆服務時，即視為服務已開始提供。
  • 服務開始提供後，將不適用《消費者保護法》第十九條所規定之七日解除權。

3. 退費原則

  • 於服務尚未開始提供前，若依法仍得行使解除權，將依相關法令規定辦理。
  • 一經用戶使用點數或系統已開始提供服務後，點數即不得申請退費或折換為現金。
  • 若因系統異常、技術錯誤或其他非可歸責於用戶之原因，導致點數扣減錯誤，經查證屬實後，本平台將返還相應點數。

4. 帳號終止與剩餘點數處理

若用戶有重大違反本平台使用規範或相關法令之情形，經合理通知仍未改善者，本平台得終止其帳號及服務。已提供並使用之服務不予退費；尚未使用之點數，將依違規情節、實際情況及相關法令規定，視個案處理。

四、隱私權與數據安全

本平台重視並致力於保護用戶之資料安全與隱私權。用戶提供之 API 資料、客戶資料及相關數據，僅用於提供與運作用戶專屬之 AI 代理人服務，不會出售、出租或以其他形式提供予未經授權之第三方，除非依法配合主管機關要求或法律另有規定。
